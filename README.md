# Задание к занятию "Внешние отчеты, обработки и расширения"

## Задача 1. Внешняя обработка «Калькулятор»

### Что сделано:
- Создана внешняя обработка с именем `Калькулятор`.
- На форме добавлены два реквизита типа `Число`: `Значение` и `Результат`.
- На форму вынесены два поля ввода.
- Добавлены 4 команды: `Сложить`, `Вычесть`, `Разделить`, `Умножить`.
- Команды вынесены на форму как кнопки.
- Для каждой команды создан обработчик `&НаКлиенте`.
- В обработчиках реализованы арифметические действия:
  - `Результат = Результат + Значение`
  - `Результат = Результат - Значение`
  - `Результат = Результат / Значение`
  - `Результат = Результат * Значение`
- Обработка сохранена в файл `Калькулятор.epf`.
